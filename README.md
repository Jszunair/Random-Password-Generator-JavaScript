# Random Password Generator

The **Random Password Generator** is a simple web application that generates secure, random passwords. This project allows users to easily create passwords using a combination of letters, numbers, and special characters.

## Features
- Generates two random passwords simultaneously.
- Supports a wide range of characters, including uppercase letters, lowercase letters, numbers, and special characters.
- User-friendly interface with a button to generate passwords.

## Technologies Used
- HTML
- CSS
- JavaScript

## Usage
1. **Open the application**: Open the `index.html` file in your web browser.
2. **Generate Passwords**: Click the "Generate" button to create two random passwords.
3. **Copy Passwords**: The generated passwords will be displayed on the page for easy copying.

## Project Structure
- `index.html`: Main HTML file containing the structure of the application.
- `style.css`: Stylesheet for the application's layout and design.
- `script.js`: JavaScript file handling the password generation functionality.
